# 智能课程表系统

基于Flask框架开发的简洁大方的课程表管理系统，支持课程添加、编辑、删除和智能提醒功能。

## 功能特点

- 简洁大方的界面设计
- 课程表的可视化展示
- 课程时间冲突检测
- 即将到来的课程提醒
- 本地数据库持久化存储（SQLite）
- 响应式设计，支持移动端访问

## 安装和运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
python app.py
```

### 3. 访问系统

在浏览器中打开：http://localhost:5000

## 使用说明

### 添加课程

1. 点击"添加课程"按钮
2. 填写课程信息（课程名称、星期、时间为必填项）
3. 点击"保存"按钮

### 编辑课程

1. 在课程表中点击要编辑的课程
2. 修改课程信息
3. 点击"保存"按钮

### 删除课程

1. 在课程表中点击要删除的课程
2. 在编辑界面中点击"删除"按钮（需要在前端添加删除功能）

### 查看即将到来的课程

点击"即将到来"标签页，查看今天剩余和明天的课程安排。

## 数据库

系统使用SQLite数据库（`course_schedule.db`）存储课程数据，首次运行时会自动创建数据库和表结构。

## 技术栈

- 后端：Flask
- 数据库：SQLite
- 前端：HTML5, CSS3, JavaScript (原生)

## 注意事项

- 系统会自动检测课程时间冲突
- 课程数据存储在本地数据库文件中
- 建议定期备份 `course_schedule.db` 文件
